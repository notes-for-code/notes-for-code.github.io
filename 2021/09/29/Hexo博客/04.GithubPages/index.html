Hexo博客-04.Github Pages

发表于 2021-09-29 更新于 2022-01-19 分类于 Hexo

配置 github pages 从而能够通过 xxx.github.io 网址访问之前自动提交到该仓库的静态文件

04.Github Pages

4.1 配置

  • 在 xxx.github.io 仓库 打开 Settings --> Pages

  • 配置 Source 下的内容并保存

    • Branch：为之前配置 hexo 的 deploy 的 branch 值

4.2 访问网站

  • 可以访问：https://xxx.github.io 访问部署的博客站点了

4.3 搞定

  • 到现在算是搞定了，后面就是本地编写 markdown 博客文件存放到 source/_posts 目录下即可
  • 提交到 github 后，github 会自动执行 actions，重新生成静态文件并提交到 xxx.github.io，稍等片刻就可以访问最新的博客站点了

发表于 2021-09-29 更新于 2021-09-29 分类于 Hexo

Hexo 的 Next 主题安装、定制配置文件及添加特殊页面:分类、标签、关于 页面

02.Next主题

2.1 配置 Hexo 主题为 Next 主题

  • 安装主题（二选一）

    1. 在站点根目录下：npm install hexo-theme-next
    2. 在站点根目录下：git clone https://github.com/next-theme/hexo-theme-next themes/next

  • 修改站点根目录下 _config.yml 文件 theme 字段值为 next

  • 重新生成静态文件并启动服务

2.2 获取配置文件

  • 根据 文档
    • 是使用 npm 安装的主题：cp node_modules/hexo-theme-next/_config.yml _config.next.yml
    • 是使用 git 从 github 仓库拉取的：cp themes/next/_config.yml _config.next.yml
  • 后面就可以根据需要修改 _config.next.yml 来定制 next 主题提供的配置项

2.4 添加菜单页面

  • 可以参考 Custom Pages

  • 添加 分类 页面：Adding «Categories» Page

    • 执行 hexo new page categories 生成页面

    • 修改 source/categories/index.md 文件

  • 添加 标签 页面：Adding «Tags» Page

    • 执行 hexo new page tags 生成页面

    • 修改 source/tags/index.md 文件

  • 添加 关于 页面：

    • 执行 hexo new page about 生成页面

    • 修改 source/about/index.md 文件

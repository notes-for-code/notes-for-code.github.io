发表于 2022-01-19 更新于 2022-01-19 分类于 Hexo

Hexo 的安装、常用命令以及一些配置项

01.Hexo 设置

1.1安装 node.js

  • 略

1.2 全局安装 Hexo

  • 执行 npm install hexo-cli -g

1.3 命令

  • 创建博客：hexo init [folder]

    • 如果不添加目录，则表示在当前目录创建

  • 启动服务：hexo server，默认是在本地 4000 端口开启

    • -p 或 --port ：设置端口

    • -s 或 --static：只使用静态文件

    • -l 或 --log：启动日记记录，使用覆盖记录格式

    • 网页效果

  • 生成静态文件：hexo generate

    • 生成的静态文件在博客目录下的 public 目录内

  • 部署：hexo deploy ，可以简写为 hexo d

1.4 设置

  • Hexo 的设置文件是博客根目录下的 _config.yml

  • Site 设置一些站点的信息

    • site：站点名字
    • language：语言，中文为 zh-CN

  • Extensions 设置扩展

    • theme：设置主题。更改主题后，需要将对应主题文件存放到 themes 目录下

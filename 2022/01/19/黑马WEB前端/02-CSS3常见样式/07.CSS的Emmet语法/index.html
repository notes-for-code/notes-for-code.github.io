  2. 快速生成 CSS 样式语法

7.1 快速生成 HTML 结构语法

  1. 生成标签：直接输入标签名按 tab 键即可。

  2. 如果想要生成多个相同的标签，加上 * 就可以了。

    • 比如 div*3 就可以快速生成三个 div

  3. 如果有父子关系的标签，可以用 >。

    • 比如 ul>li 就可以了

  4. 如果有兄弟关系的标签，用 + 就可以了。

    • 比如 div+p

  5. 如果生成电邮类名或 id 名字的，直接写 标签名.类名 或者 标签名#id 然后 tab 键就可以了

    • 比如 div.nav 就生成了 class 为 nav 的 div 元素

  6. 如果生成的 div 类名是有顺序的，可以用自增符号 $

    • 比如 div.demo$*5 就生成了 class 为 demo1-5 的 div

  7. 如果想要在生成的标签内部写内容可以用 {} 表示

    • 比如 div{$}*5 生成了内容为 1-5 的五个 div
